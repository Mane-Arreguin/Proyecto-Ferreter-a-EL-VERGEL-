# Sistema de Administración y Gestión - Ferretería EL VERGEL

Proyecto desarrollado durante la Estadía I de la carrera de Ingeniería de Datos e Inteligencia Artificial.

## Descripción

Este proyecto consiste en el desarrollo de una base de datos escalable para la administración y gestión de la Ferretería EL VERGEL. El sistema permite organizar información relacionada con productos, ventas e inventario, utilizando Python, SQL Server, Excel y Power BI.

## Tecnologías utilizadas

- Python
- Tkinter
- SQL Server
- Excel
- Power BI
- OpenPyXL
- PyODBC
